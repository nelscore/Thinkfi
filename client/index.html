❓ Quick Help Guide
Get started with ThinkFi in 4 easy steps
1. ➕ Data Input — Log income and expenses with category, date, and notes.
2. 📊 Analytics — View charts, spending breakdown, and all transactions.
3. 🎯 Goals — Set savings targets with deadlines and track progress.
4. 🤖 AI Assistant — Ask anything — powered by Claude via a secure server proxy.
💡 Pro tip: Run npm run dev to start the server, then open localhost:3000.
